# LifeAsGame

Личная экосистема для геймификации жизни: Telegram-бот собирает данные о дне,
финансах и тренировках в PostgreSQL, а Streamlit-дашборд показывает анализ и корреляции.

Статус: **Этап 1** — ежедневный чек-ин состояния.

## Возможности (этап 1)

- Ежедневный чек-ин в Telegram: время отхода ко сну и подъёма (бот сам считает длительность сна),
  тренировка, часы интенсивной работы, витамины (чеклист), настроение, энергия, оценка дня, вес, заметка.
- Хранение в PostgreSQL (источник правды).
- Напоминание в 21:00 с кнопкой «Заполнить день».
- Доступ только у владельца (`OWNER_CHAT_ID`).

## Установка

```powershell
# 1. Виртуальное окружение
python -m venv .venv
.\.venv\Scripts\activate

# 2. Зависимости
pip install -r requirements.txt

# 3. Конфиг
copy .env.example .env
# заполни DATABASE_URL, BOT_TOKEN, OWNER_CHAT_ID в .env

# 4. Создать схему и засеять справочники
python -m scripts.init_db

# 5. Запуск бота
python -m src.bot.main
```

В Telegram: `/start` → кнопка «📋 Заполнить день» (или команда `/day`). `/cancel` прерывает диалог.

## Запуск 24/7 на Windows-сервере

- Простой вариант: `scripts\run_bot.bat` + Планировщик задач Windows (триггер «При входе»/«При запуске»,
  действие — запустить bat, «Выполнять вне зависимости от регистрации пользователя»).
- Надёжный вариант: установить как службу через [NSSM](https://nssm.cc/):
  `nssm install LifeAsGameBot` → Path: `python`, Arguments: `-m src.bot.main`,
  Startup directory: корень проекта.

## Структура

```
src/config.py          конфиг из .env
src/db/                 модели, сессия, засев, CRUD (repo.py)
src/bot/                бот: main, фильтр владельца, клавиатуры, FSM, планировщик
src/bot/handlers/       common (меню), daily (чек-ин)
scripts/init_db.py      создать схему + засеять справочники
scripts/run_bot.bat     запуск бота
```
